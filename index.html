Etiquetas

La etiqueta sirve para respetar espacios y lineas en blanco, al contrario de la etiqueta parráfo

Hola soy yanina tengo 15 años 

        Hola        soy                 Yanina
                tengo           15 años

Etiqueta abbr

La etiqueta abr se utiliza para las abreviaturas

Según los datos de OMS el consumo excesivo de azúcar provoca daños severos en la salud de los niños.

Etiqueta bdo

Si se desea cambiar la direccion de un texto, se utiliza la etiqueta bdo

Este es un texto normal

Este es un texto diferente

Etiqueta cite

La etiqueta cite se utiliza para escribir citas textuales

Ojala se les duplique lo que nos deseen por: Myke Towers

La etiqueta smal hace el texto más pequeño

Este texto está dentro de una etiqueta small

Etiqueta sub

Si se desea escribir un sub índice, se debe utilizar la etiqueta sub

143 3

Etiqueta sup

Si se desea escribir u super indice se debe utilizar la etiqueta sup

1433

Etiqueta time

Para escribir fechas y horas se debe de utilizar la etiqueta time

Mañana es dia 2024-22-06

Mañana es dia 6 de noviembre del 2024

Etiqueta adress

Para especificar al navegador que el texto se refiere a una dirección utiliza la etiqueta adress

loma dorada no.100 Colonia Lomas de Rosales CP 89100 Tampico., Tamaulipas
